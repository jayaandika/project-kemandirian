# Project Summary
The Independence Assessment System is a web-based application that assesses individuals' independence in daily living activities, including Activities of Daily Living (AKS), Instrumental Activities of Daily Living (AIKS), and the Barthel Index. It collects demographic data for scoring and interpretation, aiding in determining independence levels and care eligibility. The system features an admin login, user-friendly dashboard, and cloud synchronization through Supabase for seamless data access across devices.

# Project Module Description
The project includes the following functional modules:
1. **Login System**: Admin authentication with protected routes.
2. **Dashboard**: Central hub for navigation and displaying assessment statistics.
3. **Demographic Form**: Captures user demographic information.
4. **AKS Assessment**: Evaluates daily living activities with scoring.
5. **AIKS Assessment**: Assesses instrumental activities with scoring.
6. **Barthel Index**: Measures functional independence.
7. **Assessment Result**: Displays total scores and independence levels.
8. **Assessment History**: Lists assessments with details, deletion, and export options.
9. **Theme Switcher**: Toggles between dark and light modes.
10. **Mobile Optimization**: Enhanced layout for mobile devices.
11. **Data Synchronization**: Exports and imports assessment data using JSON files.
12. **Cloud Integration**: Integrated with Supabase for real-time updates and data storage.
