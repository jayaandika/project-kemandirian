# Project Summary
The project is a web-based assessment tool aimed at evaluating individuals' independence in daily living activities. It provides assessments for Activities of Daily Living (AKS), Instrumental Activities of Daily Living (AIKS), and the Barthel Index, while collecting demographic data for comprehensive scoring and interpretation. This application assists in determining independence levels and eligibility for ongoing care, featuring an admin login and a user-friendly dashboard for effective navigation and management.

# Project Module Description
The project consists of the following functional modules:
1. **Login System**: Admin authentication with protected routes.
2. **Dashboard**: Central hub for navigation, displaying assessment statistics and management options, now includes interactive filtering for assessment history.
3. **Demographic Form**: Captures user demographic information, including health conditions and insurance details.
4. **AKS Assessment**: Evaluates six daily living activities, providing scores and interpretations of independence levels.
5. **AIKS Assessment**: Assesses eight instrumental activities, similarly scoring and interpreting results.
6. **Barthel Index**: Measures functional independence with 10 items, providing a score and interpretation.
7. **Assessment Result**: Displays total scores from all assessments, overall independence level, and PJP criteria.
8. **Assessment History**: Shows a list of all assessments conducted, with options to view details, delete entries, and export data to Excel, with new filtering capabilities based on dashboard interactions.
