# Project Summary
The Independence Assessment System is a web application that evaluates individuals' independence in daily living activities. By utilizing metrics such as Activities of Daily Living (AKS) and Instrumental Activities of Daily Living (AIKS), the system assesses user independence to determine eligibility for care. Recent enhancements include updates to the AIKS assessment format, scoring criteria, and improved data visualization features, making evaluations more comprehensive and user-friendly.

# Project Module Description
The project includes the following functional modules:
1. **Login System**: Admin authentication with protected routes.
2. **Dashboard**: Central hub for navigation and displaying assessment statistics.
3. **Demographic Form**: Captures user demographic information.
4. **AKS Assessment**: Evaluates daily living activities with a scoring system (9 functions, max score 17).
5. **AIKS Assessment**: Assesses instrumental activities with a new scoring system (8 activities, max score 8).
6. **Barthel Index**: Measures functional independence.
7. **Assessment Result**: Displays total scores and independence levels.
8. **Assessment History**: Lists assessments with details, deletion, and export options, now including percentage of independence and visual statistics.
9. **User Management**: Admin-only module for managing users (add, edit, delete).
10. **Theme Switcher**: Toggles between dark and light modes.
11. **Mobile Optimization**: Enhanced layout for mobile devices.
12. **Data Synchronization**: Exports and imports assessment data using JSON files.
13. **Cloud Integration**: Integrated with Supabase for real-time updates and data storage.
14. **Excel Export**: Exports assessment data into a structured Excel format, reflecting the new scoring criteria.
