# Project Summary
The Independence Assessment System is a web-based tool designed to evaluate individuals' independence in daily living activities. It offers assessments for Activities of Daily Living (AKS), Instrumental Activities of Daily Living (AIKS), and the Barthel Index, while also collecting demographic data for comprehensive scoring and interpretation. The application helps determine independence levels and eligibility for ongoing care, featuring an admin login and a user-friendly dashboard for easy navigation and management. The recent integration of Supabase has enabled cloud data synchronization, allowing users to access their assessment data across multiple devices seamlessly.

# Project Module Description
The project consists of the following functional modules:
1. **Login System**: Admin authentication with protected routes.
2. **Dashboard**: Central hub for navigation, displaying assessment statistics and management options, including interactive filtering for assessment history.
3. **Demographic Form**: Captures user demographic information, including health conditions and insurance details.
4. **AKS Assessment**: Evaluates six daily living activities, providing scores and interpretations of independence levels.
5. **AIKS Assessment**: Assesses eight instrumental activities, similarly scoring and interpreting results.
6. **Barthel Index**: Measures functional independence with 10 items, providing a score and interpretation.
7. **Assessment Result**: Displays total scores from all assessments, overall independence level, and PJP criteria.
8. **Assessment History**: Shows a list of all assessments conducted, with options to view details, delete entries, and export data to Excel, with filtering capabilities based on dashboard interactions.
9. **Theme Switcher**: Allows users to toggle between dark mode, light mode, and system default.
10. **Mobile Optimization**: Improved layout for mobile devices, ensuring accessibility and usability.
11. **Data Synchronization**: Features for exporting and importing assessment data using JSON files, facilitating manual data synchronization between devices.
12. **Cloud Integration**: Fully integrated with Supabase for storing assessments, enabling auto-sync across devices and real-time updates.
