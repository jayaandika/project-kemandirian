# Project Summary
The project is a web-based assessment tool designed to evaluate the independence of individuals in daily living activities. It focuses on two main assessments: Activities of Daily Living (AKS) and Instrumental Activities of Daily Living (AIKS). The application collects demographic data and provides scores and interpretations based on user input, ultimately categorizing the level of independence and determining if the individual qualifies for ongoing care (PJP).

# Project Module Description
The project consists of the following functional modules:
1. **Demographic Form**: Captures user demographic information including name, age, gender, address, and phone number.
2. **AKS Assessment**: Evaluates six daily living activities, providing scores and an interpretation of independence levels.
3. **AIKS Assessment**: Assesses eight instrumental activities, similarly scoring and interpreting results.
4. **Assessment Result**: Displays the total scores from both assessments, overall independence level, and PJP criteria.
